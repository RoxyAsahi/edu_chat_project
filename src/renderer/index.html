      学习日志

      本地学习日志关闭后 DailyNote 工具、自动协议注入和日志写入都会停用。
      作为提示变量允许 {{DailyNoteGuide}} 展开为 DailyNote 协议。
      发送前自动补上协议工具对话时自动把 DailyNote 协议追加到 system prompt。
      DailyNote 协议文本留空时使用内置默认协议。

      回答表现

      结构化渲染提示控制回答是否携带结构化渲染指导。
      表情包提示让模型知道可用表情包和图片伪路径。
      简洁气泡补充控制回答偏紧凑还是偏展开。
      视觉气泡主题注入额外追加到 system prompt 末尾的视觉风格提示。

      自动辅助

      追问提示词模板用于回复完成后生成后续追问。
      自动命名话题首轮助手回复成功后尝试自动生成话题标题。

      协议预览

      当前提示词片段查看哪些片段会被加入最终 system prompt。
      最终完整 system prompt当前选中学科实际发送给上游的完整 system prompt。

      聊天显示

      聊天字体控制普通对话内容的字体风格。
      代码字体控制代码块和等宽内容的字体。
      气泡宽度控制宽屏聊天气泡最大宽度百分比。
      平滑流式输出让模型回复逐步展开，视觉上更顺滑。

      database 来源管理

      这里专注管理来源库本身。连接与检索模型已经复用全局连接中的配置。

      每个话题会自动拥有自己的 Source；这里可统一查看和维护来源库。
