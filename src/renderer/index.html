      高频常用

      这里只保留最常改的几项：渲染提示、气泡提示、DailyNote 协议和追问模板。

      {{VarDivRender}} / {{VarRendering}} 控制回答是否携带结构化渲染指导。留空时会自动显示并使用内置默认文案。
      {{VarEmoticonPrompt}} / {{VarEmojiPrompt}} 控制模型是否知道当前内置表情包、可用文件名，以及应该输出怎样的 <img src="/通用表情包/文件名" width="120"> 伪路径。启用后会自动追加到最终 system prompt；如果当前 agent prompt 已经显式写了 {{VarEmoticonPrompt}} 或 {{VarEmojiPrompt}}，系统会跳过自动追加以避免重复。留空时会自动显示并使用内置默认文案。
      {{VarAdaptiveBubbleTip}} 控制回答偏紧凑还是偏展开。留空时会自动显示并使用内置默认文案。
      append to system prompt 这是一段真正“追加到 system prompt 末尾”的附加提示。留空时会自动显示并使用默认文案。
      DailyNote 协议文本（全局） {{DailyNoteTool}} / auto inject
      这份文本会作为 {{DailyNoteTool}}、{{VarDailyNoteGuide}} 的展开内容，也会被“自动追加到 system prompt”复用。{{StudyLogTool}} 仅作为兼容别名继续可用。留空时会直接显示并使用默认协议。
      追问提示词模板（全局） {{CHAT_HISTORY}} 留空时会自动显示并使用内置默认模板。若未显式写入 {{CHAT_HISTORY}}，系统会把裁剪后的最近对话自动追加到模板末尾。
      {{CHAT_HISTORY}} 只会在首轮助手回复成功后尝试自动命名。留空时会自动显示并使用内置默认模板，要求模型只返回 {\"title\":\"😀 标题\"} 这种 JSON。
